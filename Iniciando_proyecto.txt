1.- git init

2.- git status  --> muestra todos los cambios

3.- git add . --> Agregando todos los archivos.

4.- git commit -m "Datos uno"  --> hace la primer copia.

5.-git commit -am "Sube y guarda el commit"   -->  sube y guarda el commit

6.-git log  --> muestra los commits

7.-Agregar diferentes formas.

git add index.html 

git add *.png  

git add css/  //todo lo que esta en la carpeta
